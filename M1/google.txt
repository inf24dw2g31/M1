## Steps to Authorize Swagger UI:

1. First, get your JWT token by visiting: http://localhost:3000/auth/token
   - This endpoint will return JSON with your token like `{"token": "eyJhbGciOiJ..."}`

2. In the Swagger UI page (http://localhost:3000/docs):
   - Click the "Authorize" button at the top right (it looks like a lock)
   - In the "bearerAuth (JWT)" section, enter your token in the format: `Bearer your_token_here`
   - Click "Authorize" then "Close"

3. Now your API requests should include the authentication token
